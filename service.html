Front End

Front end adalah bagian dari sebuah aplikasi atau situs web yang berinteraksi langsung dengan pengguna. Komponen-komponennya meliputi HTML untuk struktur, CSS untuk tampilan, JavaScript untuk interaktivitas, serta desain UI/UX dan responsivitas. Ini adalah bagian yang pengguna lihat dan gunakan.

Back End

Backend adalah bagian dari aplikasi web yang bekerja di belakang layar untuk mengelola data, logika bisnis, dan komunikasi dengan server serta database. Ini melibatkan penggunaan bahasa pemrograman server-side, kerangka kerja backend, database, API, serta implementasi keamanan dan logika bisnis. Ini adalah bagian yang tidak terlihat langsung oleh pengguna tetapi penting untuk menjalankan aplikasi dengan lancar.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat aplikasi web interaktif. Ini dapat dijalankan di browser untuk mengatur tampilan dan perilaku halaman web, serta di server untuk mengelola logika bisnis aplikasi.
